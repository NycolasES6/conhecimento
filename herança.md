# Herança

### Definição

Herança é a capacidade de uma classe filha derivar ou herdar as características e comportamentos da classe pai. Assim fornece a reutilização de código. Pode ser aplicado em [[Python]], [[Java]] e etc....

## Sintaxe

### Python

```python
class A:
	pass

class B(A):
	pass

```

## Herança múltipla

### Python

```python
class A:
	pass

class B:
	pass

class C(A, B):
	pass

```
